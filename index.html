1. Introdução

================================
| NÃO COMERCIALIZE ESSE SCRIPT |
================================

COMBINAR TECLAS 2.0, por Reynolds Costa

Essa biblioteca tem a finalidade de tornar possível o uso de sequências de teclas para executar funções.

2. Instalação

Para começar a usar essa biblioteca, copie os arquivos combinarTeclas2.0_func.js e combinarTeclas2.0_main_mini.js* na pasta js do presente repositório e faça conexões com eles no seu html usando o atributo src da tag <script>.

A sua instalação deve se parecer com isso:

Ver código

<script type = "text/javascript" language = "JavaScript" src = "js/combinarTeclas2.0_main_mini.js"></script>

<script type = "text/javascript" language = "JavaScript" src = "js/combinarTeclas2.0_main_func.js"></script>

* Não é necessário usar o arquivo combinarTeclas2.0_main.js, mais pesado que sua contraparte minificada, disponível nesse repositório apenas para consulta e contribuição.

3. Utilização

Abaixo da conexão descrita no item anterior, basta instanciar a classe CombinarTeclas em uma constante CT segundo a sintaxe abaixo. Mova o mouse sobre o trecho para saber mais.

Ver código

const CT = new CombinarTeclas([

[arr1, function () {

// CÓDIGO DA FUNÇÃO 1

}],

[arr2, function () {

// CÓDIGO DA FUNÇÃO 2

}],

[arr3, function () {

// CÓDIGO DA FUNÇÃO 3

}],

// ETC

]);

Observe o código de exemplo desta página:

Ver código

// ARQUIVO DE EXEMPLO

 

const CT = new CombinarTeclas([ // Declaração do objeto

// Ctrl+Enter chama a função ola();

[[17, 13], ola],

// Ctrl+Shift+Enter chama a função soma();

[[17, 16, 13], function() {

alert(soma([1,2])); // Exibe o resultado de soma([1, 2]);

}]

]);

 

// A função ola()

function ola() {

alert("Olá, mundo!");

}

 

// A função soma(parcelas)

function soma(parcelas) {

let produto = 0;

parcelas.forEach((parcela) => {

produto += parcela

});

return produto;

}

4. Palavras reservadas

  • Como classe JavaScript: CombinarTeclas
